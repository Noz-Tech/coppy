# Coppy Release Notes

## v0.1.0-alpha

`v0.1.0-alpha` is the first public alpha of Coppy, a secure personal vault for IDs, cards, policies, and other private records.

### Highlights

- Save and organize private entries in one place, including IDs, cards, and other personal records.
- Group entries into folders for faster browsing and cleaner organization.
- Search entries quickly from the home screen.
- Copy individual values or copy a full entry in one tap.
- Hide sensitive entries and reveal them only when needed.

### Security

- Optional biometric lock when opening the app.
- Optional biometric confirmation before revealing values.
- Optional biometric confirmation before copying data.
- Optional biometric confirmation before sharing data.
- Optional biometric protection before showing hidden items.

### Experience

- First-launch welcome flow for new users.
- Sample data to help users understand how entries are structured.
- Settings screen for security preferences and privacy controls.
- Built-in options for Terms & Conditions, Data Privacy, and About Coppy.
- Full local vault reset for wiping saved entries and folders from the device.

### Platform Notes

- Android app package updated to `org.noztek.coppy`.
- Built with Kotlin Multiplatform for Android and iOS.

### Alpha Notes

- This is an early alpha release focused on core vault management and biometric protection.
- Users may still encounter rough edges while the app experience and feature set continue to mature.
